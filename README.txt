【READ ME】

1.説明・概要

    [1.vscode-portable.bat]
・このソフトは、vscode.zipを用いて、自身のvscode環境を外出先PCに反映するソフトです。

・このリポジトリは主にbatファイルを使用しており、２種類の機構を有しています。
    -1.普段使用のvscodeからデータ(extensionやuser_info)をコピーする-
    -2.データをvscode-portable上で展開し、新PCでの環境構築を図る-

2.必要なもの

・USB
→できればGen3.1以上が好ましい
→空き容量は1GBあればOK

3.使い方

    1.このvscode-portableパッケージをUSB直下にダウンロードする
    (このとき、フォルダに「1.build-env / 2.vscode.zip」が揃っているか確認してください)

    2.vscode-portable\build-envへ移動し、[vscode-portable.bat]を起動しましょう

    3.聞かれた質問にぞれぞれ回答し、3~5分ほど時間をおけば、完了です。


-おすすめの使い方-

・外出前にimportでデータを取り込んでおく
(日頃からこまめに取り込んでおくのも良いでしょう)

・必要な時に自動でデータを出力し、手動では別作業にリソースが割ける！
→新PCでの環境構築がスムーズに行えるでしょう。
